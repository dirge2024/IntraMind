## 2. 接口设计

```
embedding/contract/
└── EmbeddingService
    └── List<float[]> embed(List<String> texts)

embedding/impl/
├── QwenEmbeddingService      # 通义千问 text-embedding-v3
├── DeepSeekEmbeddingService  # DeepSeek OpenAI 兼容

```

## 3. 接口抽象

```java
public interface EmbeddingService {
    /**
     * @param texts 文本列表（单次最多 25 条）
     * @return 向量列表，每个 float[2048]
     */
    List<float[]> embed(List<String> texts);
}
```

## 5. API 调用（OkHttp）

```java
// Qwen
POST https://dashscope.aliyuncs.com/api/v1/services/embeddings/text-embedding/text-embedding
Headers: Authorization: Bearer {apiKey}, Content-Type: application/json
Body: {"model":"text-embedding-v3","input":{"texts":["text1","text2"]},"parameters":{"dimension":2048}}
Response: {output: {embeddings: [{embedding: [0.1, 0.2, ...]}, ...]}}

// DeepSeek (OpenAI 兼容)
POST https://api.deepseek.com/v1/embeddings
Headers: Authorization: Bearer {apiKey}, Content-Type: application/json
Body: {"model":"deepseek-embedding","input":["text1","text2"]}
Response: {data: [{embedding: [0.1, 0.2, ...]}, ...]}
```

## 6. EmbeddingConsumer 批量处理

```java
@KafkaListener(topics = "document.chunked", groupId = "localrag-embedding")
public void onMessage(String json) {
    payload = deserialize(json);
    chunks = payload.getChunks();

    for (int i = 0; i < chunks.size(); i += BATCH_SIZE) {
        batch = chunks.subList(i, min(i + BATCH_SIZE, chunks.size()));
        texts = batch.stream().map(ChunkData::getText).toList();

        vectors = embeddingService.embed(texts);  // 最多 3 次重试

        for (int j = 0; j < batch.size(); j++) {
            batch.get(j).denseVector = vectors.get(j);
        }
    }

    messageProducer.send("embedding.requested", EmbeddingRequestedPayload.builder()
        .md5(payload.getMd5())
        .chunks(chunks)
        .build());
}
```

## 7. 重试策略

```java
private List<float[]> embedWithRetry(List<String> texts) {
    for (int attempt = 1; attempt <= 3; attempt++) {
        try {
            return embeddingService.embed(texts);
        } catch (Exception e) {
            if (attempt == 3) throw e;
            Thread.sleep((long) Math.pow(2, attempt - 1) * 1000);
        }
    }
}
```
