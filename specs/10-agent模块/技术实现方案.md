## 3. 实现策略

### 3.1 ReAct 循环核心算法

```java
// ReActLoop.java — 核心循环
GenerationHandle chat(String userId, String query, String sessionId,
                      Consumer<AgentEvent> eventSink) {

    String generationId = UUID.randomUUID().toString();
    generationStateManager.create(generationId, userId, sessionId);
    eventSink.accept(new StartEvent(generationId, sessionId));

    List<Map<String, Object>> messages = buildMessages(query, sessionId);
    int executedToolCalls = 0;
    int totalPromptTokens = 0, totalCompletionTokens = 0;

    for (int round = 1; round <= config.getMaxRounds(); round++) {
        if (generationStateManager.isCancelled(generationId)) break;

        List<AgentTool> tools = (round == config.getMaxRounds() || executedToolCalls >= config.getMaxToolCalls())
                ? List.of() : toolRegistry.getTools();

        ReActTurn turn = streamReActTurn(userId, messages, tools, generationId, eventSink);
        if (turn == null) break; // 流被取消

        totalPromptTokens += turn.promptTokens();
        totalCompletionTokens += turn.completionTokens();

        if (turn.toolCalls().isEmpty()) {
            // 模型决定直接回答
            complete(generationId, userId, query, turn, eventSink, totalPromptTokens, totalCompletionTokens);
            return handle;
        }

        // 执行工具调用
        messages.add(turn.assistantMessage());
        for (ToolCallDecision tc : turn.toolCalls()) {
            eventSink.accept(new ToolCallEvent(generationId, tc.name(), "executing", tc.id()));

            if (executedToolCalls >= config.getMaxToolCalls()) {
                messages.add(toolMessage(tc.id(), "工具调用预算已用尽"));
                eventSink.accept(new ToolCallEvent(generationId, tc.name(), "failed", tc.id()));
                continue;
            }

            ToolResult result = executeTool(userId, query, generationId, tc, eventSink);
            messages.add(toolMessage(tc.id(), result.content()));
            executedToolCalls++;

            if (result.streamedToUser()) {
                complete(generationId, userId, query, turn, eventSink, totalPromptTokens, totalCompletionTokens);
                return handle;
            }
        }
    }

    // 预算耗尽，强制最终回答
    messages.add(Map.of("role", "user", "content", "请基于已有结果给出最终回答，不要调用工具。"));
    ReActTurn finalTurn = streamReActTurn(userId, messages, List.of(), generationId, eventSink);
    complete(generationId, userId, query, finalTurn, eventSink, totalPromptTokens, totalCompletionTokens);
    return handle;
}
```

### 3.2 流式 ReAct Turn（WebClient + 短轮询取消）

```java
// ReActLoop.java
private ReActTurn streamReActTurn(String userId, List<Map<String, Object>> messages,
                                  List<AgentTool> tools, String generationId,
                                  Consumer<AgentEvent> eventSink) {
    CompletableFuture<ReActTurn> future = new CompletableFuture<>();
    StreamAccumulator acc = new StreamAccumulator();

    WebClient client = buildWebClient(llmProvider);
    Disposable sub = client.post()
            .uri("/chat/completions")
            .bodyValue(buildReActRequest(messages, tools, config.getMaxCompletionTokens()))
            .retrieve()
            .bodyToFlux(String.class)
            .subscribe(
                    chunk -> {
                        acc.consume(chunk);
                        if (!acc.getContentDelta().isEmpty()) {
                            eventSink.accept(new ChunkEvent(generationId, acc.getContentDelta()));
                        }
                    },
                    future::completeExceptionally,
                    () -> future.complete(acc.toTurn())
            );

    activeStreams.put(generationId, new StreamHandle(sub, () -> settleUsage(acc)));

    // 短轮询等待，每200ms检查取消标记
    long deadline = System.nanoTime() + TimeUnit.SECONDS.toNanos(config.getCompletionTimeout());
    try {
        while (true) {
            if (generationStateManager.isCancelled(generationId)) {
                sub.dispose();
                return null;
            }
            long remaining = deadline - System.nanoTime();
            if (remaining <= 0) {
                sub.dispose();
                throw new TimeoutException("LLM 响应超时");
            }
            try {
                return future.get(Math.min(200, TimeUnit.NANOSECONDS.toMillis(remaining)), TimeUnit.MILLISECONDS);
            } catch (TimeoutException ignored) { }
        }
    } catch (InterruptedException e) {
        Thread.currentThread().interrupt();
        sub.dispose();
        return null;
    } finally {
        activeStreams.remove(generationId);
    }
}
```

### 3.3 Tool Call Delta 流式聚合算法

```java
// StreamAccumulator.java
class StreamAccumulator {
    private final StringBuilder content = new StringBuilder();
    private final Map<Integer, StreamingToolCall> toolCalls = new LinkedHashMap<>();

    void consume(String sseLine) {
        JsonNode node = objectMapper.readTree(sseLine);
        JsonNode delta = node.path("choices").path(0).path("delta");

        // 文本 delta
        String text = delta.path("content").asText("");
        if (!text.isEmpty()) {
            contentDelta = text;
            content.append(text);
        }

        // tool_calls delta（分片到达，按 index 聚合）
        JsonNode tcArray = delta.path("tool_calls");
        for (JsonNode tc : tcArray) {
            int idx = tc.path("index").asInt(toolCalls.size());
            StreamingToolCall stc = toolCalls.computeIfAbsent(idx, i -> new StreamingToolCall());
            String id = tc.path("id").asText("");
            if (!id.isEmpty()) stc.id = id;
            String name = tc.path("function").path("name").asText("");
            if (!name.isEmpty()) stc.name = name;
            String args = tc.path("function").path("arguments").asText("");
            if (!args.isEmpty()) stc.arguments.append(args);
        }
    }

    ReActTurn toTurn() {
        List<ToolCallDecision> decisions = new ArrayList<>();
        for (StreamingToolCall stc : toolCalls.values()) {
            Map<String, Object> args = objectMapper.readValue(stc.arguments.toString(), new TypeReference<>(){});
            decisions.add(new ToolCallDecision(stc.id, stc.name, args));
        }
        return new ReActTurn(content.toString().trim(), decisions, finishReason, promptTokens, completionTokens);
    }
}
```

### 3.4 Tool Registry 注册机制

```java
// AgentToolRegistry.java
@Component
public class AgentToolRegistry {

    private final Map<String, AgentTool> tools = new LinkedHashMap<>();

    public AgentToolRegistry(RetrievalService retrievalService, LlmProvider llmProvider) {
        register(new SearchKnowledgeTool(retrievalService));
        register(new GenerateSummaryTool(retrievalService, llmProvider));
        register(new KnowledgeStatsTool());
    }

    void register(AgentTool tool) {
        tools.put(tool.name(), tool);
    }

    List<Map<String, Object>> getOpenAiTools() {
        return tools.values().stream().map(tool -> Map.of(
                "type", "function",
                "function", Map.of(
                        "name", tool.name(),
                        "description", tool.description(),
                        "parameters", tool.parameters()
                )
        )).toList();
    }

    ToolResult execute(String name, Map<String, Object> args, String userId) {
        AgentTool tool = tools.get(name);
        if (tool == null) return new ToolResult("未知工具: " + name, Map.of(), false);
        return tool.execute(args, userId);
    }
}
```

### 3.5 工具实现示例

```java
// SearchKnowledgeTool.java
@Component
public class SearchKnowledgeTool implements AgentTool {

    private final RetrievalService retrievalService;

    @Override
    public String name() { return "search_knowledge"; }

    @Override
    public String description() { return "在知识库中检索与查询相关的文档片段，返回多个候选结果"; }

    @Override
    public Map<String, Object> parameters() {
        return Map.of(
                "type", "object",
                "properties", Map.of(
                        "query", Map.of("type", "string", "description", "检索关键词或自然语言查询"),
                        "topK", Map.of("type", "integer", "description", "返回结果数量，默认5")
                ),
                "required", List.of("query")
        );
    }

    @Override
    public ToolResult execute(Map<String, Object> args, String userId) {
        String query = (String) args.getOrDefault("query", "");
        int topK = args.get("topK") instanceof Number n ? n.intValue() : 5;
        List<RetrievalResult> results = retrievalService.search(query, topK);
        if (results.isEmpty()) {
            return new ToolResult("知识库中未找到相关内容。", Map.of("results", List.of()), false);
        }
        String content = formatResults(results);
        return new ToolResult(content, Map.of("results", results), false);
    }

    private String formatResults(List<RetrievalResult> results) {
        StringBuilder sb = new StringBuilder("检索到 ").append(results.size()).append(" 条结果：\n");
        for (int i = 0; i < results.size(); i++) {
            RetrievalResult r = results.get(i);
            sb.append("[").append(i + 1).append("] ");
            if (r.getPageNumber() != null && r.getPageNumber() > 0) {
                sb.append("（").append(r.getFileName()).append(" | 第").append(r.getPageNumber()).append("页）");
            } else {
                sb.append("（").append(r.getFileName()).append("）");
            }
            sb.append(" ").append(truncate(r.getText(), 300)).append("\n");
        }
        return sb.toString();
    }
}
```

```java
private void buildReferenceMapping(String generationId, List<RetrievalResult> results) {
    Map<Integer, ReferenceInfo> mapping = new HashMap<>();
    for (int i = 0; i < results.size(); i++) {
```

### 3.7 生成状态生命周期（Redis）

```java
// GenerationStateManager.java
@Component
public class GenerationStateManager {

    void create(String generationId, String userId, String sessionId) {
        String metaKey = "chat:generation:" + generationId + ":meta";
        redisTemplate.opsForHash().putAll(metaKey, Map.of(
                "userId", userId,
                "conversationId", sessionId,
                "status", "STREAMING",
                "createdAt", Instant.now().toString()
        ));
        redisTemplate.expire(metaKey, Duration.ofMinutes(30));
    }

    void appendChunk(String generationId, String chunk) {
        redisTemplate.opsForValue().append(
                "chat:generation:" + generationId + ":content", chunk);
    }

    void markCompleted(String generationId, Map<Integer, ReferenceInfo> refs) {
        setStatus(generationId, "COMPLETED");
        if (refs != null && !refs.isEmpty()) {
            redisTemplate.opsForValue().set(
                    "chat:generation:" + generationId + ":refs",
                    objectMapper.writeValueAsString(refs),
                    Duration.ofMinutes(30));
        }
    }

    void markCancelled(String generationId) { setStatus(generationId, "CANCELLED"); }
    void markFailed(String generationId, String error) {
        setStatus(generationId, "FAILED");
        redisTemplate.opsForHash().put(
                "chat:generation:" + generationId + ":meta", "error", error);
    }
}
```

### 3.8 上下文压缩策略

```java
// ReActLoop.java
private List<Map<String, Object>> buildMessages(String query, String sessionId) {
    List<Map<String, Object>> messages = new ArrayList<>();

    // 系统 Prompt（可配置）
    messages.add(Map.of("role", "system", "content", promptBuilder.buildSystemPrompt()));

    // 对话历史：仅保留最近 6 条，单条截断 800 字符
    List<ChatHistoryMessage> history = chatHistoryManager.getRecent(sessionId, 6);
    for (ChatHistoryMessage msg : history) {
        String content = msg.getContent().length() > 800
                ? msg.getContent().substring(0, 800) + "..."
                : msg.getContent();
        messages.add(Map.of("role", msg.getRole(), "content", content));
    }

    // 当前问题
    messages.add(Map.of("role", "user", "content", query));
    return messages;
}
```

## 9. 线程模型

```
Tomcat Thread (HTTP)
  │ 接收 GET /api/llm/chat
  │ 提交 Runnable 到 chatExecutor
  │ 立即返回（异步 DeferredResult 或 SSE Emitter）
  └──→ chatExecutor Thread
        │ 执行 ReActLoop.runReActLoop()
        │ 每轮 streamReActTurn() → WebClient 异步 IO
        │ 收到 chunk → eventSink → SSE 输出
        │ 循环直到完成/取消/超时
        └── 清理 activeStreams + 结算 Token
```

`chatExecutor` 配置：core=4, max=8, queueCapacity=50, rejectedPolicy=CallerRunsPolicy。
