# 前端页面 — 需求分析

## 目标结构

### 页面路由

```
/                    → index.html     控制台
/upload.html         → upload.html   分片/直传上传
/kb.html             → kb.html       知识库文件列表
/chat.html           → chat.html     对话助手
/sessions.html       → sessions.html  历史会话管理
```

## 统一设计规范

### 侧边栏（5 页完全一致，抽取为 common.css）

```
侧边栏：240px 固定宽，白底圆角卡片（16px），左侧留白 16px
  ├─ 品牌区：logo + "LocalRAG"，底部 1px 分隔线
  ├─ 导航区（4 个平级入口）：
  │   ├─ 📄 文件上传
  │   ├─ 📚 知识库
  │   ├─ 💬 对话助手
  │   └─ 🕐 历史会话
  └─ （仅对话助手页） [+ 新对话] 按钮
```

### 主内容区

| 页面 | 内容 | 特殊 |
|------|------|------|
| index | 流水线状态 + ES/MinIO 卡片 | — |
| upload | 分片模式切换 + 文件选择 + 进度条 + 日志 | — |
| kb | 文件表格（文件名/MD5/大小/状态/时间/操作） | — |
| chat | 消息流 + 底部固定输入框 + 左下角连接状态灯 | 聊天区适当增大 |
| sessions | 搜索框 + 日期筛选 + 会话卡片列表 | — |

### 公共组件

| 组件 | 说明 | 文件 |
|------|------|------|
| 侧边栏样式 | 所有页共享 | **common.css**（新建） |
| 自定义确认弹窗 | showConfirm() | 各页面 script 内 |
| 状态标签 | .badge-* | kb.html |
| 连接状态灯 | 绿色/红色圆点 + 文字 | chat.html |

### 连接状态灯（chat.html）

```
左下角：🟢 已连接 / 🔴 未连接
  - 页面加载时 GET /api/health → 200 → 绿色
  - 失败或超时 → 红色
  - SSE 流中断 → 红色
  - 恢复 → 绿色
  - 轻量实现：30s 轮询一次 health 接口
```
