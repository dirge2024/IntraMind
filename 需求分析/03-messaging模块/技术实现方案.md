# Messaging 模块 — 技术实现方案

## 1. 依赖

- `org.springframework.kafka:spring-kafka`（Spring Boot 托管版本）
- `com.fasterxml.jackson.core:jackson-databind`（Spring Boot 托管）
- `com.localrag:common`

## 2. DTO 放哪

`FileUploadedPayload` 既被 storage 发消息用，又被 document 消费用。但 storage 不依赖 messaging（当前依赖链：storage 独立）。两种方案：

| 方案 | 做法 | 优点 | 缺点 |
|------|------|------|------|
| A | DTO 放 common | 谁都能用 | common 膨胀 |
| B | storage 依赖 messaging | 符合模块职责 | 改现有依赖 |

> **选方案 A**。`FileUploadedPayload` 只是一个数据载体，放 common 的 `dto` 包里，后续其他模块的消息 Payload 也放 common。消息体 `KafkaMessage<T>` 留在 messaging。

## 4. 接口

```java
public interface MessageProducer {
    <T> void send(String topic, T payload);
}
```

## 5. 实现

```java
@RequiredArgsConstructor
public class KafkaMessageProducer implements MessageProducer {
    private final KafkaTemplate<String, String> kafkaTemplate;
    private final ObjectMapper objectMapper;

    @Override
    public <T> void send(String topic, T payload) {
        KafkaMessage<T> message = KafkaMessage.of(topic, payload);
        String json = objectMapper.writeValueAsString(message);
        kafkaTemplate.send(topic, json).addCallback(
            result -> log.debug("kafka sent: topic={}, offset={}", topic, result.getRecordMetadata().offset()),
            ex -> log.error("kafka send failed: topic={}", topic, ex)
        );
    }
}
```

## 6. Kafka 配置

```yaml
spring:
  kafka:
    bootstrap-servers: ${KAFKA_BROKERS:localhost:9092}
    producer:
      key-serializer: org.apache.kafka.common.serialization.StringSerializer
      value-serializer: org.apache.kafka.common.serialization.StringSerializer
    consumer:
      key-deserializer: org.apache.kafka.common.serialization.StringDeserializer
      value-deserializer: org.apache.kafka.common.serialization.StringDeserializer
      group-id: localrag-${spring.application.name}
      enable-auto-commit: false
```

## 7. 消息体结构

```java
@Data
public class KafkaMessage<T> {
    private String messageId;    // UUID
    private String topic;
    private String timestamp;    // ISO 8601
    private T payload;

    public static <T> KafkaMessage<T> of(String topic, T payload) {
        KafkaMessage<T> msg = new KafkaMessage<>();
        msg.messageId = UUID.randomUUID().toString();
        msg.topic = topic;
        msg.timestamp = Instant.now().toString();
        msg.payload = payload;
        return msg;
    }
}
```

## 10. 集成点

storage 模块 `StorageController.completeUpload()` 和 `directUpload()` 完成后调用：

```java
messageProducer.send("document.uploaded", FileUploadedPayload.builder()
    .md5(md5)
    .fileName(...)
    .fileSize(...)
    .objectKey(...)
    .build());
```

storage 需要新增对 common 的依赖（已有），以及注入 `MessageProducer`。
